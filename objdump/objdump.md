#一、objdump -h
##1.1 demo.cpp

```cpp
#include <iostream>
using namespace std;
 
int main()
{
	int count = 0;
	printf("result: %d\n", count);
	return 0;
}
```
编译

gcc -c demo.cpp [-lstdc++]

生成一个demo.o文件，即obj文件

#二、 objdump -D
##2.1 main.c

```c
#include <stdio.h>

void fun(int a)
{
}
 
int main()
{
    int a = 1234;
    fun(a);
}
```
##2.2 反汇编代码main.asm如下
objdump -D main.o >main.asm
反汇编代码main.asm如下
